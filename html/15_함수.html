함수(function)

        * 목표하는 결과를 만들기 위한 코드의 묶음

        * 코드를 {} (블록)으로 묶어 작성하고 붙여진 이름을 호출해서 사용

        - 함수 선언
        - 함수 정의
        - 함수 호출
        
        [기본 함수 형태]

        function 함수명(){  // 함수 선언

            // 함수 정의

        }

        함수명() ; // 함수 호출

매개 변수(parameter), 전달 인자(Arguments)

        * 매개 변수
            - 함수 선언 시 () 안에 작성되는 변수
                전달 인자로부터 받은 값을 저장하고 함수 정의 부분에서 사용 가능
        
        * 전달 인자
            - 함수 호출 시 () 안에 작성되는 값
                함수 매개 변수로 전달되는 값

참조

        * 객체 지향 언어에서 배열, 객체 등 object만 여러개를 가지고 있는 자료형인데
        변수는 값을 1개만 저장할 수 있기 때문에

        object를 메모리 다른 곳에 저장하고 저장된 곳의 주소를 변수에 저장해서
        필요할 때 마다 해당 주소를 찾아가 원하는 값을 얻어오거나 수정할 수 있게 하는 것

        이 때, 주소를 찾아가는 것을 '참조'라고 한다.

Return(반환)

        * 현재 수행중인 함수를 종료하고 호출한 곳으로 돌아감
        * 함수 호출 구조는 Stack 형식으로 되어있다.
         -> FILO(First In Last Out, 선입후출) 구조
        * 호출한 곳으로 돌아갈 때 return 옆에 작성된 값을 가지고 돌아갈 수 있다.
        ex) return 100; // 호출한 곳으로 100을 가지고 돌아감
        * 함수 마지막에 return 구문이 생략되어 있다면 자동으로 return; 구문이 추가

익명 함수(이름이 없는 함수)

        [기본함수]
        function 함수명([매개변수]){}   

        함수명(); 이름이 있으니까 함수명으로 호출 가능


        [익명함수]
        function([매개변수]){}  

        // 함수 이름이 없어서 호출할 수 없다!

        기본 함수와는 다르게 변수에 대입, 매개 변수로 바로 전달되어야 하는 곳에 주로 사용

        ex) const sumFn = function(a,b){return a+b;} 
        ex2) element.addEventListener("click",function(){})

화살표 함수(Arrow Function)

        - 익명 함수의 한 종류
        - 익명 함수를 간단히 표기하는 방법 (ES6)
    
        익명 함수 :  function(){}
    
        1. 기본 형태 :  () => {}
    
    
        
        2. 매개변수 1개 :  (e) => {}  또는  
                            e  => {}  // () 생략
    
    
        3. 매개변수 n개 :  (a,b,c) => {}   //  () 생략 불가
    
    
        4. 함수 정의 부분에 return 한 줄만 있을 경우 :
    
          1) 익명 함수 :               function(){ return 1 + 2; }
    
          2) 화살표 함수(기본) :       () => { return 1 + 2; }
           
          3) 화살표 함수(return 한줄) : () => 1 + 2
                                        // {return} 생략
    
    
        5. 함수 정의 부분에 return 한 줄만 있는데
          return하는 값이 object(객체, 배열 등) 인 경우
          {return} 생략 불가!!!
    
          () => { return {name:"홍길동", age:20}; }




즉시 실행 함수

        ( ()=>{} )()
        -> 익명 함수를 바로 수행하는 함수

        --- 즉시 실행 함수를 사용하는 이유 ---
        
        1) 성능(속도)적 우위를 가지기 위해서
            -> 아주 약간 빠름

        2) 변수명 중복 문제를 해결하기 위해서

        const num1 = 100; // 전역 변수
        (() => {
            const num1 = 200; // 지역 변수

            console.log(num1); // = 200
        })()
